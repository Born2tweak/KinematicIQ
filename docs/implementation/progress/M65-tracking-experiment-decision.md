# M65 — Evidence-gated tracking experiment and adoption decision

**Status:** Complete — no candidate adopted (2026-07-11).

M64 found no synchronized external reference that can measure a filter/model
improvement while detecting event lag or peak attenuation. UI-PRMD reduced data
are not MediaPipe-paired; the OCHuman real corpus is unavailable; local tapes
have exact rep counts but no bottom-frame MAE.

Under M65's predeclared rule, “rejected/no change” is a valid completion outcome.
No filter, MediaPipe option, confidence rule, rep gate, threshold, version, or
pose-tape provenance changed. The existing default stays unchanged.

Reopen M65 only with a saved before baseline and a compatible reference suite.
The comparison must report event lag, waveform error, per-trial regressions,
exact rep count, camera e2e, and runtime budget.
