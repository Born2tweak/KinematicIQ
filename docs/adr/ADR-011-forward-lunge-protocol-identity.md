# ADR-011: Forward-Lunge Protocol Identity

**Status:** Accepted for documentation; implementation migration pending P4-M01.

## Decision

Use `Forward Lunge` in product copy and `Forward lunge with stride and return` in scientific material. The canonical protocol ID is `forwardLungeStrideReturn`, task version is `forward-lunge-stride-return-v1`, and observation protocol is `side-view-forward-lunge-stride-return-v1`.

The implementation at `f49558e` performs a step, descent, bottom, ascent, and return; it is not the fixed-stance FMS inline lunge. This ADR does not change code or availability.
